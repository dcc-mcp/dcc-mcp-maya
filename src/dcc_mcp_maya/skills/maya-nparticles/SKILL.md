# maya-nparticles

Maya nParticle skill. Provides actions for creating and managing Maya nParticle systems.

## Scripts

- `create_nparticle_system` — Create a new nParticle system with nucleus solver
- `list_nparticle_systems` — List all nParticle objects in the scene
- `set_nparticle_attribute` — Set an attribute on an nParticle object
- `emit_particles` — Emit particles from a surface or volume
- `delete_nparticle_system` — Delete an nParticle system
